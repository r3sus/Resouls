dehook DLL

hack to make NohBoard and YAKD input visualisers work with Dead Space games

mechanism: stubs the USER32.SetWindowsHookExA, no side effects found

usage:

put the dll in game folder, launch the game normally
- chainload feature:
to use with mouse fix or other DLLs
add to it's name _load_ or put in _load_ folder
or use external chainloader to load both DLLs.

for temporary effect or testing, use CT:
https://github.com/r3sus/cheads/ds_/dehook
