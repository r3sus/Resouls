ds1/r mowin dll

dll for manipulating dark souls 1/r game window on start

by default, it moves game window to left top corner

customization
_mowin.txt is created with 5 arguments (2-6 of SetWindowPos API)
default is all 0
Z - set to -1 for always-on-top
X, Y - position
cx, cy - size (don't use)

use with any chainloader
